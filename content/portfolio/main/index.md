# 팀 프로젝트

저는 "사람들이 즐겁게 플레이하는 게임을 만들어보자" 라는 목표로 직접 사람들을 모집해 팀을 만들거나, 이미 있는 팀에 들어가 작업하는 경험을 쌓아왔습니다.

* 총 3개의 팀 프로젝트에서 활동하였고, 그 중 2개의 프로젝트에서 **팀장**을 맡았습니다.
* 모든 개발 프로젝트에서 **메인 프로그래머**로 작업하였고, 프로젝트의 주요 기술적 의사결정들을 주도하였습니다.
